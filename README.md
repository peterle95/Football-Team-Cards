# Football-Team-Cards

An interactive web application that displays player cards for the 1986 Argentina World Cup-winning football team. Users can filter players by position or by those with nicknames.
Features

Team Information Display: Shows team name, sport, year, and head coach
Player Cards: Displays detailed player information including position, number, and nickname
Filtering Options:

- All Players
- Players with Nicknames
- Filter by Position (Forward, Midfielder, Defender, Goalkeeper)


Responsive Design: Works seamlessly on desktop and mobile devices

## Project Structure

- index.html - HTML structure and static player card markup
- script.js - JavaScript logic for filtering and dynamic card generation
- styles.css - Styling and responsive layout

## How It Works
The application uses a JavaScript object to store team and player data. When users select a filter option from the dropdown menu, the page dynamically renders only the matching player cards using the filter() and map() array methods.

### Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
